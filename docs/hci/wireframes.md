## Wireframing Rationale
1. Controls live in a dedicated left rail to separate decision-making from browsing.
2. Recommendation cards dominate the main canvas because that is the user’s primary task.
3. The featured movie block gives one strong suggestion before the user scans the full grid.
4. Two explicit recommendation modes reduce ambiguity and support different discovery behaviors.
